# Secure Image Steganography with Hybrid Cryptography

**Current Status:** Phase 1 Complete (Foundation & Key Infrastructure)  
**Team:** 4 Members  
**Tech Stack:** Java 21, Maven, AtlantaFX (UI)

---

## 1. Project Overview
This tool secures sensitive text data using a "Defense in Depth" approach. It combines **Adaptive Encryption** (Caesar/VigenÃ¨re/AES) with **Image Steganography** (LSB Embedding) and protects key exchange using **Elliptic Curve Cryptography (ECC)**.

---

## 2. Current Implementation (What works now)
We have established the "Clean Start" architecture:

* **File Input & Basic Encryption:**
    * `src/main/java/com/stego/FileEncryptor.java`
    * Reads text files, splits logic, and applies encryption based on word length.
* **ECC Key Infrastructure:**
    * `src/main/java/com/stego/ECCManager.java`
    * Generates valid `secp256r1` Key Pairs.
    * Handles Base64 Key Encoding/Decoding for the UI.

## 3. Setup Instructions (For Teammates)
**Prerequisites:**
1.  **Java 21 (LTS)** installed.
2.  **VS Code** with the "Extension Pack for Java".

**How to Run:**
1.  Clone this repository.
2.  Open the folder in VS Code.
3.  Wait for Maven to build the project.
4.  Run `src/main/java/com/stego/FileEncryptor.java` to test the logic.
